# Aulas sobre Layouts no HTML5 e CSS3

Nesta aula são apresentados diversos exemplos de como elaborar layouts utilizando o HTML e CSS.

- Na pasta **layouts-css** você encontrá 4 exemplos de layouts constrídos com: Float, Position, Grid Layout e Flex Box.

- Na pasta **ex-mediaquerie** você encontrá um exemplo de layout responsivo construído por meio de @media do CSS.

- Na pasta **ex-bootstrap** você encontrá um exemplo de layout responsivo construído utilizando o Framework Bootstrap.

- Na pasta **ex-layout** você encontrá um exemplo de layout contruído utilizando alguns dos conceitos apresentados anteriormente.
